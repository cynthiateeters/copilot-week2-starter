# Week 2: Use Agent mode

Welcome to Week 2! This assignment teaches you to use Copilot Agent for project setup while you learn npm basics through the fun cowsay package.

## Getting started

1. Open this folder in VS Code
2. Read [docs/INSTRUCTIONS.md](docs/INSTRUCTIONS.md) for the full assignment
3. Follow the parts in order

- Use Agent mode for setup tasks
- Understand npm and package.json
- Edit JSON files manually
- Run npm scripts
- Create documentation with Agent
